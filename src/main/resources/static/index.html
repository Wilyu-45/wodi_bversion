游戏规则

游戏简介

谁是卧底是一款多人互动推理游戏。玩家中平民获得相同的词语，而卧底获得与平民相似的词语。天使获得二者的词语。白板不获得词语，只有提示词。 每轮玩家用简短的词语描述自己手中的词语，不可重复，会有相应的限制词存在。

游戏流程

  1. 系统为每位玩家分发对应的词语，平民与卧底玩家不知道自己的身份
  2. 所有玩家轮流描述自己的词语
  3. 每轮描述后，所有玩家投票选出可能是卧底与白板的人
  4. 得票最多的玩家出局，如果出现平票，则无人出局
  5. 在夜晚阶段，认为自己是卧底的玩家进行刀人，如果他确实是卧底，则目标出局，如果他不是卧底，则自己出局

胜利条件

  • 平民胜利：成功投出卧底与白板
  • 卧底胜利：存活到最后
  • 白板胜利：投出卧底后存活并猜对平民与卧底的词语
